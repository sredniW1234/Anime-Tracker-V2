# Anime-Tracker-V2
This app is for tracking which anime/shows/movies/books/manga you've seen. 

**Supported Items: Anime/Shows, Movies,** *Manga/Books (WIP)*

# Current Features:
### Items:
* Both Season and Movie items.
* Status Selection
* Rating
* Favoriting
* Genre Selection
* Notes
* Auto Tracking
* Dates Started & Ended
* Automatic Tracking of weekly, bi-weekly, or monthly episodes (WIP & NEEDS TESTING)
* Icon Selection

### General:
* Searching
* Fast View Switching
* List Stats (WIP)
* Simple Shortcuts (new, save, save as, load, and focus [F])
* Filtering by Status
* Quick Collapse

There probably will be bugs, so sorry about that!

## Shortcuts
* New     - Ctrl+N
* Save    - Ctrl+S
* Save As - Ctrl+Alt+S
* Open    - Ctrl+O
* Focus   - F
